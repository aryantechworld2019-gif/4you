# React + Tailwind Starter (Vite)


## Quick start


1. Install dependencies


```bash
npm install
```


2. Start dev server


```bash
npm run dev
```


3. Build for production


```bash
npm run build
```


## Notes & tips
- This template uses Vite. If you prefer Create React App, replace the build tooling.
- Tailwind is configured via `tailwind.config.cjs`. Add your component paths to `content` when you add new folders.
- For TypeScript: rename files to `.tsx` and add `typescript` devDependency + `tsconfig.json`.
